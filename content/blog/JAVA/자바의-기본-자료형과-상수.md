## 자바의 기본 자료형
자바는 총 8개의 기본 자료형을 가지고 있다.

1. 정수 자료형: byte, short, int, long
- 4개의 정수 자료형이 갖는 차이점은 __정수의 표현에 사용하는 메모리 공간의 크기에 있다.__ 
  
- byte는 1바이트, short는 2바이트, int는 4바이트, long은 8바이트를 기준으로 정수를 표현한다.  

> 참고! 자바는 정수형 연산을 4바이트 int형으로 진행한다.  
```java
short result = short num1 + short num2 ; // 컴파일 에러
```  

---
2. 실수 자료형: float, double
- __실수 자료형은 정밀도를 포기하고 표현의 범위를 넓힌 자료형이기 때문에 기본적으로 계산 오차가 발생한다.__   
  
- float는 소수점 이하 6자리까지 오차가 발생하지 않고, double은 소수점 이하 15자리까지 오차가 발생하지 않는다. 하지만 오차가 존재하는 데이터 둘 이상을 연산하다 보면 더 일찍 오차가 발생할 수도 있다.  

---
3. 문자 자료형: char
- 자바는 __유니코드(unicode)를 기반으로__ 문자를 처리한다. 하지만 문자의 출력은 OS에 의존적이기 때문에, 출력하고자 하는 언어의 폰트가 해당 OS에 설치되어 있어야 문자가 정상적으로 출력된다.

> 참고! String은 기본 자료형이 아니다!  

```java
char ch1 = '확';
```  

---
4. 논리 자료형: boolean
- boolean에 참을 의미하는 값인 true와 거짓을 의미하는 값인 false를 저장할 수 있다.

```java
System.out.println(10<5); // false 출력
System.out.println(10>5); // true 출력
```

## 자바의 상수
final 선언을 추가하면 그 변수는 상수가 된다.
- __상수는 값을 딱 한 번만 할당할 수 있다.__
- 한 번 할당된 값은 변경이 불가능하다.

## 상수의 표기법
1. 상수의 이름은 모두 __대문자__로 짓는다.
2. 이름이 둘 이상의 단어로 이뤄질 경우 단어 사이에 __언더바__를 넣는다.

```java
final int MAX = 100;
final int CONST_ASSIGNED;
CONST_ASSIGNED = 12;
```

## 마무리
1. 자바의 기본 자료형은 8개`(byte, short, int, long, float, double, char, boolean)`이다.
2. 실수 자료형의 연산에서 `오차`가 발생할 수 있다.
3. 자바의 상수는 `final` 선언으로 만들어진다.
4. 상수의 이름은 모두 `대문자`로 짓는다.
